DEFINITION LIST

Definition list is among the types of list in HTML. The list includes:

Ordered
The content is listed in numbers, roman, alphabets etc..as below:
  1. 1
  2. A
  3. a
  4. I
  5. i
Unordered
Definition
Definition/Description list (dl) has an element describing tag ( dd ) is used to describe a term/name in a description list. It starts a line more to the right describing the description tag/term.

Definition list has neither bullets,letters nor numbers

Nested
